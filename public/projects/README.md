# Project screenshots

Drop screenshots into the matching folder, then list them in `src/data/projects.ts`
under each project's `images` array. Example:

```ts
images: [
  "/projects/devops-pipeline/dashboard.png",
  "/projects/devops-pipeline/pipeline.png",
],
```

Folders ready for you:
- `devops-pipeline/`
- `gym-management/`
- `senior-care/`
- `food-delivery/`
